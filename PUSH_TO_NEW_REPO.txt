=================================================================
  PUSH TO NEW GITHUB REPOSITORY - SIMPLE COMMANDS
=================================================================

STEP 1: Create a new repository on GitHub
-----------------------------------------
1. Go to https://github.com/new
2. Create a new repository (e.g., "task-management-system")
3. DO NOT initialize with README, .gitignore, or license
4. Copy the repository URL


STEP 2: Remove old git history (if exists)
-----------------------------------------
rmdir /s /q .git


STEP 3: Initialize fresh git repository
-----------------------------------------
git init
git branch -M main


STEP 4: Add your new remote repository
-----------------------------------------
git remote add origin https://github.com/YOUR_USERNAME/YOUR_REPO_NAME.git

Example:
git remote add origin https://github.com/johndoe/task-management-system.git


STEP 5: Verify no sensitive data will be pushed
-----------------------------------------
git status

# Check for credentials (should return nothing)
git add .
git diff --cached | findstr /i "zeus91356 Shivamsati password secret"

If you see any results, STOP and fix them first!


STEP 6: Commit and push
-----------------------------------------
git add .
git commit -m "Initial commit: Enterprise Task Management System"
git push -u origin main
